About

One engineer. A fleet of agents. A shipping product.

I'm an AI-native product engineer based in the U.S. I build production software by treating AI coding agents, LLM tooling, and automation as a coordinated engineering system — not a copilot. My role is architect, orchestrator, and quality bar: I design the spec, choose the models, manage the context, and ship the output.

Under that approach, I built the entire Sasquatch Wars codebase solo: 290 files, 20+ game systems, real-time multiplayer, cross-platform distribution (Web/Steam/iOS/Android), and a full Xsolla monetization stack (26 SKUs live). Active build time: ~30 hours. Output quality: a live, playable, paying-user product.

This isn't a demo of what AI can do. It's a demonstration of what an engineer directing AI correctly can do — at a velocity and scope that previously required a 10-person team.

Capability

How I ship at this velocity.

Shipping a 290-file full-stack product solo isn't about "AI writing the code." It's about applying engineering discipline to an emergent medium — agent orchestration, context management, evaluation, and spec design. Below is the skill stack behind the output.

⚙

Agent Orchestration

Designing multi-step, multi-tool agent workflows with persistent state, shared workspaces, and parallel subagents. I treat each agent as a callable function with a spec — not a chatbot.

🧠

Model Selection & LLM Systems

Picking the right model for each step — image generation, code, reasoning, transcription — based on cost, quality, and latency. Building LLM-powered features into shipped products (Xsolla catalog, content generation, game copy).

📝

Spec & Prompt Engineering

Turning ambiguous product goals into executable checklists, structured skills, and reusable prompt templates. Every ambiguity removed at spec time is an hour saved at build time.

🚀

Full-Stack Shipping Velocity

React · TypeScript · Express · SQLite · Drizzle · Vite · Tailwind · Capacitor · Electron · Render. One brain, one codebase, four distribution channels, one live payment processor.

📊

Systems & Product Thinking

Financial modeling (Excel with 367 live formulas), monetization design, Kickstarter campaign architecture, launch playbooks, and brand systems. I build the business, not just the code.

🎯

Quality & Evaluation

I set the quality bar. Agents execute. I review, redirect, and re-spec until the output meets production standard — the same discipline a staff engineer applies to a team's PRs.

Real-Time Strategy
